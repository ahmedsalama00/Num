Bisection Method Root Finding

Interval halving method. Requires f(a)·f(b) < 0 on [a,b].

Condition: \( f(x) \) must be continuous on \([a,b]\) and \( f(a) \cdot f(b) < 0 \). Number of iterations: \( n \geq \dfrac{\ln\left(\frac{b-a}{\varepsilon}\right)}{\ln 2} \)
Input Parameters
Use: x^2, 3*x^2, sin(x), cos(x), exp(x), log(x), sqrt(x), Math.PI

False Position Method Root Finding

Uses a secant line instead of midpoint. Faster convergence than bisection.

Formula: \( x_s = \dfrac{a \cdot f(b) - b \cdot f(a)}{f(b) - f(a)} \). Requires \( f(a) \cdot f(b) < 0 \).
Input Parameters
Use: x^2, sin(x), cos(x), exp(x), log(x), sqrt(x)
Function Plot
Convergence Animation

Newton-Raphson Method Root Finding

Uses f(x) and its derivative f'(x). Fast convergence near the root.

Formula: \( x_{n+1} = x_n - \dfrac{f(x_n)}{f'(x_n)} \). Method fails if \( f'(x_n) \approx 0 \).
Input Parameters
Use: x^2, sin(x), cos(x), exp(x), log(x), sqrt(x) — derivative computed automatically
Auto-Derivative: \( f'(x) \) is computed numerically using central difference: \( f'(x) \approx \dfrac{f(x+h) - f(x-h)}{2h} \)
Function Plot & Iteration Path
Convergence Animation

Secant Method Root Finding

Like Newton-Raphson but without needing the derivative analytically.

Formula: \( x_{n+1} = x_n - \dfrac{f(x_n)(x_n - x_{n-1})}{f(x_n) - f(x_{n-1})} \). Requires two initial points.
Input Parameters
Use: x^2, sin(x), cos(x), exp(x), log(x), sqrt(x)

Jacobi Iteration Linear Systems

Method of simultaneous displacements. Requires diagonal dominance: |a_ii| > Σ|a_ij|.

Condition: \( |a_{ii}| > \sum_{j \neq i} |a_{ij}| \) (strict diagonal dominance). All initial values set to 0.

Gauss-Seidel Method Linear Systems

Method of successive approximations. Uses updated values immediately — faster than Jacobi.

Key difference from Jacobi: Each \( x_i \) uses the most recent computed values. Condition: \( |a_{ii}| \geq \sum_{j \neq i} |a_{ij}| \).

Doolittle constraint: \( L_{ii} = 1 \). Solves \( Ax = b \) by forward + backward substitution.

Newton Forward Interpolation Interpolation

Use when target x is near the beginning of the table. Equally spaced data required.

Formula: \( f(x) = y_0 + p\,\Delta y_0 + \dfrac{p(p-1)}{2!}\Delta^2 y_0 + \cdots \) where \( p = \dfrac{x - x_0}{h} \)
Data Points (equally spaced)
2 to 8 points
Interpolation Plot

Newton Backward Interpolation Interpolation

Use when target x is near the end of the table. Uses backward difference table.

Formula: \( f(x) = y_n + p\,\nabla y_n + \dfrac{p(p+1)}{2!}\nabla^2 y_n + \cdots \) where \( p = \dfrac{x - x_n}{h} \)
Data Points (equally spaced)
Interpolation Plot

Stirling Central Interpolation Interpolation

Use when target x is near the center of the table. Central difference formula.

Formula: \( f(x) = y_0 + p\cdot\dfrac{\Delta y_0 + \Delta y_{-1}}{2} + \dfrac{p^2}{2!}\Delta^2 y_{-1} + \cdots \) where \( p = \dfrac{x - x_0}{h} \), \( x_0 \) is center.
Data Points (odd count, equally spaced)
Interpolation Plot

Lagrange Interpolation Interpolation

Works for unequally spaced data. Can also be used for inverse interpolation.

Formula: \( f(x) = \sum_{i} L_i(x) \cdot y_i \) where \( L_i(x) = \prod_{j \neq i} \dfrac{x - x_j}{x_i - x_j} \). No equal spacing required.
Data Points (any spacing)
Interpolation Plot

Forward Differentiation Differentiation

Based on Newton's Forward Interpolation. Use when the target point is near the beginning of the table.

f'(x0): \( \dfrac{1}{h}\left[\Delta y_0 - \frac{1}{2}\Delta^2 y_0 + \frac{1}{3}\Delta^3 y_0 - \cdots\right] \)  |  f''(x0): \( \dfrac{1}{h^2}\left[\Delta^2 y_0 - \Delta^3 y_0 + \frac{11}{12}\Delta^4 y_0 - \cdots\right] \)
Data Points (equally spaced)

Backward Differentiation Differentiation

Based on Newton's Backward Interpolation. Use when the target point is near the end of the table.

f'(xn): \( \dfrac{1}{h}\left[\nabla y_n + \frac{1}{2}\nabla^2 y_n + \frac{1}{3}\nabla^3 y_n + \cdots\right] \)  |  f''(xn): \( \dfrac{1}{h^2}\left[\nabla^2 y_n + \nabla^3 y_n + \frac{11}{12}\nabla^4 y_n + \cdots\right] \)
Data Points (equally spaced)

Stirling Differentiation Differentiation

Based on Stirling's Central Interpolation. Use when the target point is near the center of the table.

f'(x0): \( \dfrac{1}{h}\left[\dfrac{\Delta y_{-1}+\Delta y_0}{2} - \dfrac{1}{6}\cdot\dfrac{\Delta^3 y_{-2}+\Delta^3 y_{-1}}{2} + \cdots\right] \)  |  f''(x0): \( \dfrac{1}{h^2}\left[\Delta^2 y_{-1} - \frac{1}{12}\Delta^4 y_{-2} + \cdots\right] \)
Data Points (equally spaced)

Lagrange Differentiation Differentiation

Based on Lagrange's Interpolation. Works for unequally spaced data.

Method: Build the Lagrange polynomial f(x), differentiate analytically to get f'(x) and f''(x). Works for any spacing.
Data Points (any spacing)

Trapezoidal Rule Integration

Approximates the integral using straight-line segments between data points.

Formula: \( \int_a^b f(x)\,dx = \dfrac{h}{2}\left[y_0 + 2(y_1 + y_2 + \cdots + y_{n-1}) + y_n\right] \), where \( h = \dfrac{b-a}{n} \)
Input Parameters
Use: x^2, sin(x), cos(x), exp(x), log(x), sqrt(x)
Integration Plot

Simpson's 1/3 Rule Integration

Approximates the integral using parabolic curves. More accurate than Trapezoidal.

Formula: \( \int_a^b f(x)\,dx = \dfrac{h}{3}\left[y_0 + 4(y_1+y_3+y_5+\cdots) + 2(y_2+y_4+y_6+\cdots) + y_n\right] \). Condition: \( n \) must be even.
Input Parameters
Use: x^2, sin(x), cos(x), exp(x), log(x), sqrt(x)
Integration Plot

Simpson's 3/8 Rule Integration

Uses cubic polynomials. Useful when n is a multiple of 3.

Formula: \( \int_a^b f(x)\,dx = \dfrac{3h}{8}\left[y_0 + 3(y_1+y_2+y_4+y_5+\cdots) + 2(y_3+y_6+\cdots) + y_n\right] \). Condition: \( n \) must be a multiple of 3.
Input Parameters
Use: x^2, sin(x), cos(x), exp(x), log(x), sqrt(x)
Integration Plot

Boole's Rule Integration

5-point closed Newton-Cotes formula using a degree 4 polynomial. Requires n = 4.

Formula: \( \int_a^b f(x)\,dx = \dfrac{2h}{45}\left[7y_0 + 32y_1 + 12y_2 + 32y_3 + 7y_4\right] \). Condition: \( n = 4 \) (5 points).
Input Parameters
Use: x^2, sin(x), cos(x), exp(x), log(x), sqrt(x)
Integration Plot

Weddle's Rule Integration

7-point closed Newton-Cotes formula using a degree 6 polynomial. Requires n = 6.

Formula: \( \int_a^b f(x)\,dx = \dfrac{3h}{10}\left[y_0 + 5y_1 + y_2 + 6y_3 + y_4 + 5y_5 + y_6\right] \). Condition: \( n = 6 \) (7 points).
Input Parameters
Use: x^2, sin(x), cos(x), exp(x), log(x), sqrt(x)
