Bisection Method Root Finding

Interval halving method. Requires f(a)·f(b) < 0 on [a,b].

Condition: f(x) must be continuous on [a,b] and f(a)·f(b) < 0 (opposite signs). Iterations: n ≥ log((b-a)/ε) / log(2)
Input Parameters
Use Hint: x^2, 3*x^2, sin(x), cos(x), exp(x), log(x), sqrt(x), Math.PI

False Position Method Root Finding

Uses a secant line instead of midpoint. Faster convergence than bisection.

Formula: x_s = (a·f(b) - b·f(a)) / (f(b) - f(a)). Requires f(a)·f(b) < 0.
Input Parameters
Use Hint: x^2, 3*x^2, sin(x), cos(x), exp(x), log(x), sqrt(x)

Newton-Raphson Method Root Finding

Uses f(x) and its derivative f'(x). Fast convergence near the root.

Formula: x_{n+1} = x_n - f(x_n)/f'(x_n). Requires f'(x) to be computed. Method fails if f'(x_n) ≈ 0.
Input Parameters
Use Hint: x^2, 3*x^2, sin(x), cos(x), exp(x), log(x), sqrt(x)
Manually enter the derivative of f(x)

Secant Method Root Finding

Like Newton-Raphson but without needing the derivative analytically.

Formula: x_{n+1} = x_n - f(x_n)·(x_n - x_{n-1}) / (f(x_n) - f(x_{n-1})). Requires two initial points.
Input Parameters
Use Hint: x^2, 3*x^2, sin(x), cos(x), exp(x), log(x), sqrt(x)

Jacobi Iteration Linear Systems

Method of simultaneous displacements. Requires diagonal dominance: |a_ii| > Σ|a_ij|.

Condition: |a₁₁| > |a₁₂| + |a₁₃|, etc. All initial values set to 0. Converges if diagonally dominant.

Gauss-Seidel Method Linear Systems

Method of successive approximations. Uses updated values immediately — faster than Jacobi.

Key difference from Jacobi: In Gauss-Seidel, each x_i uses the most recent computed values. Condition: |a_ii| ≥ Σ|a_ij|.

Doolittle constraint: L_ii = 1. Solves Ax = b by forward + backward substitution.

Newton Forward Interpolation Interpolation

Use when target x is near the beginning of the table. Equally spaced data required.

Formula: f(x) = y₀ + p·Δy₀ + p(p-1)/2!·Δ²y₀ + ... where p = (x - x₀)/h
Data Points (equally spaced)
2 to 8 points

Newton Backward Interpolation Interpolation

Use when target x is near the end of the table. Uses backward difference table.

Formula: f(x) = y_n + p·∇y_n + p(p+1)/2!·∇²y_n + ... where p = (x - x_n)/h
Data Points (equally spaced)

Stirling Central Interpolation Interpolation

Use when target x is near the center of the table. Central difference formula.

Formula: f(x) = y₀ + p·(Δy₀+Δy₋₁)/2 + p²/2!·Δ²y₋₁ + ... where p = (x - x₀)/h, x₀ is center.
Data Points (odd count, equally spaced)

Lagrange Interpolation Interpolation

Works for unequally spaced data. Can also be used for inverse interpolation.

Formula: f(x) = Σ L_i(x)·y_i where L_i(x) = Π (x - x_j)/(x_i - x_j). No equal spacing required.
Data Points (any spacing)

Forward Differentiation Differentiation

Based on Newton's Forward Interpolation. Use when the target point is near the beginning of the table.

f'(x₀): (1/h)[Δy₀ − ½Δ²y₀ + ⅓Δ³y₀ − ¼Δ⁴y₀ + …]  |  f''(x₀): (1/h²)[Δ²y₀ − Δ³y₀ + (11/12)Δ⁴y₀ − …]
Data Points (equally spaced)

Backward Differentiation Differentiation

Based on Newton's Backward Interpolation. Use when the target point is near the end of the table.

f'(xₙ): (1/h)[∇yₙ + ½∇²yₙ + ⅓∇³yₙ + ¼∇⁴yₙ + …]  |  f''(xₙ): (1/h²)[∇²yₙ + ∇³yₙ + (11/12)∇⁴yₙ + …]
Data Points (equally spaced)

Stirling Differentiation Differentiation

Based on Stirling's Central Interpolation. Use when the target point is near the center of the table.

f'(x₀): (1/h)[(Δy₋₁+Δy₀)/2 − (1/6)(Δ³y₋₂+Δ³y₋₁)/2 + …]  |  f''(x₀): (1/h²)[Δ²y₋₁ − (1/12)Δ⁴y₋₂ + …]
Data Points (equally spaced)

Lagrange Differentiation Differentiation

Based on Lagrange's Interpolation. Works for unequally spaced data. Leave x blank to show polynomial only.

Method: Build the Lagrange polynomial f(x), differentiate analytically to get f'(x) and f''(x). Works for any spacing.
Data Points (any spacing)
